# assessment-tg-bot
### This telegram bot gives answers to questions about assessment

Deployed on Heroku. Useful commands:

    heroku login
    heroku create assessment-tg-bot
    git push heroku main
    heroku ps:scale worker=1
    heroku ps
    heroku logs --tail
